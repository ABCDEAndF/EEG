
Each folder's name indicate the specific data augmentation method

every folder relies on the dataloading folder. Except for the slicing window approach that relies on the specific two files inside the dataloading files, all other methods use the dataLoader_formatRaw and dataLoader_train-valid-test.

The baseline is main_training_CNN (first 15-subject samples as the original project did)
and main_training_CNN_full_dataset. (full dataset).
